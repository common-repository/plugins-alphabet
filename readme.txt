Adds an alphabet index on plugins page. Supports multisite !

== Description ==

Once activated the plugin adds an alphabet index on plugins page that allows you to find any plugin that starts with the letter of your choice.

[GitHub repository](https://github.com/TweetPressFr/plugins-alphabet)

[Follow me on Twitter](http://twitter.com/intent/user?screen_name=tweetpressfr)

== Installation ==

1. Upload plugin files to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Enjoy
